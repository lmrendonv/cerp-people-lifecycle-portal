Admin Guide — CERP People Lifecycle Portal
How to use this tool
Personnel Lifecycle Flow
New Joinee → Onboarding → Active Member → Exit Process → Alumni
Onboarding a new member
  1. Go to Onboarding in the sidebar
  2. Click New Onboarding
  3. Fill in name, KAUST ID, type, and supervisor
  4. Review the auto-generated checklist
  5. Click Create Onboarding Case
  6. Track progress by clicking View on any case
Starting an offboarding
  1. Go to Offboarding in the sidebar
  2. Click New Case
  3. Select the member and set departure date
  4. Click View to open the exit checklist
  5. Mark tasks as complete as they are finished
  6. Ensure Alumni profile is created before closing
